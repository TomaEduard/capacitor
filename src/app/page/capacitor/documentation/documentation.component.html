Install Capacitor on project: https://capacitorjs.com/solution/angular



ng add @capacitor/angular


npm install @capacitor/cli@latest @capacitor/core@latest


npx cap init

  • What is the name of your app?
  • What should be the Package ID for your app?(com.company.example)

ng build --prod


  • In root/capacitor.config.ts
  • config.webDir: 'dist/[name of folder]'



Install Capacitor for android & mac

npm i @capacitor/ios @capacitor/android @capacitor/electron


npx cap add android


npx cap add ios


npx cap add electron





sync dist/[project name] -> android/ios app

npm cap sync





package.json -> scripts

"run-android": "ng b --prod && npx cap copy && npx cap open android",


"run-ios": "ng b --prod && npx cap copy && npx cap open ios",


"build-ios": "ng b --prod && npx cap copy",



"start:electron": "ng build --base-href ./ --prod && npx cap copy && npx cap open electron",

"setup-win": "electron-packager . [name of project(folder name in dist/[example])] --overwrite --asar --platform=win32 --arch=ia32 --prune=true --electron-version=13.1.3 --out=release-builds --version-string.CompanyName=MyCompany --version-string.FileDescription=Test --version-string.ProductName=\"[name of project(folder name in dist/[example])]\",

"run-electron": "ng b --base-href ./ --prod && npx cap copy && npx cap open electron"
